
# Big Data em Business Intelligence

- Definição: Big Data é o termo usado para descrever grandes volumes de dados — estruturados, semiestruturados e não estruturados — gerados em alta velocidade e provenientes de fontes diversas (como redes sociais, sensores IoT, logs, aplicativos, e-commerce, ERPs e CRMs).

- Em BI, o Big Data amplia a capacidade analítica tradicional, permitindo extrair insights em tempo real e compreender comportamentos complexos que não caberiam apenas em bancos de dados convencionais.

# As 5 Características Fundamentais (Os “5 Vs” do Big Data)

- Volume:
Quantidade massiva de dados gerados.
Milhões de registros de clientes, transações ou sensores.

- Velocidade
Rapidez com que os dados são criados e processados.
Dashboards em tempo real e alertas automáticos.

- Variedade
Diferentes formatos e fontes de dados.
Dados de planilhas, APIs, vídeos, textos e IoT.

- Veracidade
Confiabilidade e qualidade dos dados.
Filtrar ruídos e inconsistências antes de analisar.

- Valor
Capacidade de gerar conhecimento e decisões.
Converter dados brutos em estratégias e resultados.

- Em resumo:Big Data só é útil quando transforma volume em valor — e o BI é a ponte que faz isso acontecer.

# Como o Big Data se conecta ao BI

- O Business Intelligence tradicional trabalha com dados estruturados (SQL, Data Warehouse).
Já o Big Data traz dados não estruturados (textos, cliques, vídeos, logs, etc.), exigindo novas tecnologias e arquiteturas.

- Fluxo integrado:

Fontes (ERP, CRM, Web, IoT)
     ↓
Data Lake (armazenamento bruto)
     ↓
Processamento (Hadoop, Spark)
     ↓
Data Warehouse / BI (análise e dashboards)

- Assim, o Analista de BI pode consumir dados tratados pelo Big Data para criar visões estratégicas em Power BI, Tableau, ou ferramentas equivalentes.

# Principais Tecnologias Relacionadas
- Armazenamento e Processamento: Hadoop, Spark, Hive, Databricks, Azure Synapse, AWS Redshift.
- Streaming e Tempo Real: Kafka, Flink, Azure Stream Analytics.
- Integração com BI: Power BI, Tableau, Qlik Sense (conectores nativos a Big Data).
- Linguagens de apoio: Python, SQL, Scala.
